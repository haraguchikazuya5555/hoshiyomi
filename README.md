# 星詠み(ほしよみ)— 毎日の運勢占いWebアプリ

生年月日と名前から、数秘術×タロット×ラッキーカラーで「今日だけの運勢」を占うWebアプリ。
静的ファイルのみで動作します(サーバー・DB・API不要、通信費ゼロ)。

## ページ構成

- `index.html` — 占いアプリ本体(今日の運勢)
- `unmeisu.html` — 運命数1〜33の解説(SEO・AdSense審査用コンテンツ、計算機つき)
- `tarot.html` — タロット大アルカナ22枚の意味(正位置・逆位置・今日のヒント)
- `assets/site.css` / `assets/stars.js` — 解説ページ共通のスタイルと星空背景

占い結果の「運命数」「タロットカード」から各解説ページの該当セクションへ直接リンクしており、
解説ページ側のCTAボタンでアプリ本体へ戻す回遊構造になっています(SEO流入→占い体験→シェアの導線)。

## 特徴(売れる設計のポイント)

- **毎日結果が変わる** … 名前+生年月日+日付をシードにした決定論的乱数。同じ人は同じ日なら必ず同じ結果 →「ちゃんと占ってる感」と毎日のリピート訪問を両立
- **入力の記憶** … localStorageで前回の名前・生年月日を自動復元。2回目以降はワンタップ
- **シェア導線** … X(Twitter)とLINEの共有ボタン内蔵。結果に絵文字+ハッシュタグ付きで拡散されやすい
- **スマホ最適化** … 占いユーザーの大半はスマホ。モバイルファーストのレイアウト
- **完全クライアント処理** … 個人情報を一切送信しない(=プライバシーポリシーが軽く済む)

## ローカルで見る

```sh
cd hoshiyomi
python3 -m http.server 8791
# → http://localhost:8791
```

またはindex.htmlをダブルクリックするだけでも動きます。

## デプロイ(無料)

- **GitHub Pages**: リポジトリにpushして Settings → Pages で公開(独自ドメイン可)
- **Cloudflare Pages / Netlify / Vercel**: フォルダをドラッグ&ドロップで完了

## 収益化ロードマップ

1. **独自ドメイン取得**(年1,000円前後)→ AdSense審査に必須級
2. **コンテンツ拡充** … 運命数解説・タロット解説ページは実装済み。さらに増やすなら「今月の運勢」「誕生日占い」など
3. **Google AdSense** … 結果画面の下(share-rowの上)と、解説ページの記事間が最適位置
4. **物販アフィリエイト** … ラッキーアイテム・パワーストーンを楽天/Amazonアフィリエイトリンクに(相性が非常に良い)
5. **LINE公式アカウント** … 「毎朝の運勢を受け取る」導線 → 濃いリピーター化
6. **有料メニューへの導線** … ココナラ等の個別鑑定、noteの有料記事(例:2026年下半期の運勢)

## 法務メモ

- フッターに「娯楽目的」の免責を記載済み
- 広告を貼る場合はプライバシーポリシーページを追加すること
- 個人情報は端末外に出ないため、現状は特定商取引法・個情法の重い対応は不要

## カスタマイズのしやすさ

占いテキストはすべて `index.html` 内の `TAROT` / `LIFE_PATH` / `*_COMMENTS` / `LUCKY_*` / `ADVICE` 配列にまとまっています。文言を増やすほど「日替わり感」が強くなります。
