## 3. GitHub Pages 側の設定(購入したら私に言ってください、こちらで実行します)

1. リポジトリに `CNAME` ファイル(中身はドメイン名1行)をコミット
2. Settings → Pages → Custom domain に入力 → Enforce HTTPS をON
3. サイト内の全URL(canonical / OGP / sitemap / robots / シェアリンク)を新ドメインに一括置換
4. Search Console に新ドメインのプロパティを追加(旧URLはアドレス変更ツールで引き継ぎ)
5. GA4 のストリームURLを更新

※3〜5は購入の報告をもらえれば全部こちらで自動実行できます。

## 4. AdSense申請(ドメイン設定後)

事前条件は整備済み:
- [x] プライバシーポリシー(privacy.html)
- [x] 独自コンテンツ(運命数解説・タロット解説の長文2ページ+占い機能)
- [x] sitemap.xml / 構造化データ
- [ ] 独自ドメイン ← ここだけ待ち

申請手順:
1. https://adsense.google.com → サイトを追加(新ドメイン)
2. 審査コード(`<script>`1行)を貼る → **コードをもらえれば私が全ページに実装してpushします**
3. ads.txt が発行されたら同様にこちらで設置
4. 審査は数日〜2週間。落ちても「コンテンツ追加→再申請」が普通(1回で通る方が少ない)
