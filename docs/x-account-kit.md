# X専用アカウント開設キット

アカウント作成とログインはご本人の操作が必要です(電話番号/メール認証があるため)。
作成後、以下をコピペすれば5分でプロフィールが完成します。

## ハンドル候補(15文字制限内・上から推奨順)

1. `@hoshiyomi_app`(13字)— サービス名+種別で検索に強い
2. `@hoshiyomi365`(12字)— 「毎日」の含意がコンセプトと一致
3. `@hoshiyomi_jp`(12字)
4. `@hoshiyomi_day`(13字)

※取得時に空き確認を。ドメインを `hoshiyomi365.com` にするならハンドルも `@hoshiyomi365` に揃えると覚えやすい。

## 運用開始チェックリスト

- [ ] アカウント作成(メールは普段使いと分けると管理が楽)
- [ ] プロフィール設定(上記コピペ)
- [ ] 固定ポスト投稿
- [ ] 毎朝の定型投稿を7日続ける(docs/marketing.md のテンプレ)
- [ ] 週1でGA4の utm_source=x 流入を確認(効果測定)

## 新規アカウントの注意点

- 開設直後は1日の投稿を3〜5件までに(スパム判定回避)
- フォロー先は占い・個人開発界隈を1日10件程度まで
- 最初の1〜2週はインプレッションが伸びなくて正常。GA4の流入だけ見る
